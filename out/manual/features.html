SyntaxHighlighter Features

  • Blogger integration
  • Collapsing
  • CSS Themes
  • Dynamic brush loading
  • Extensible brush system
  • First line number
  • HTML with script tags
  • In-line copy and paste
  • Inline URL detection
  • Line highlighting
  • No-CSS fallback
  • Smart tabs
  • Strict & XHTML compliant
  • The gutter
  • Toolbar
