This configuration option allows you to localize your highlighted elements or just change the displayed strings. The following options are available:

expandSource : "+ expand source"

help : "?"

alert : "SyntaxHighlighter\n\n"

noBrush : "Can't find brush for:"

brushNotHtmlScript : "Brush wasn't made for html-script option:"

viewSource 2.x : "view source"

copyToClipboard 2.x : "copy to clipboard"

copyToClipboardConfirmation 2.x : "The code is in your clipboard now"

print 2.x : "print"

Example

<script type="text/javascript">
    SyntaxHighlighter.config.strings.viewSource = "view my source!!!!";
</script>
