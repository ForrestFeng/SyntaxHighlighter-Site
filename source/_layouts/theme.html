How to use?

Picking and using a theme is as easy as linking to an external CSS file. All you have to do is provide the correct path and the file name of theme that you want to use. Alternatively, you can embed the theme that you want on the download page. 

<link type="text/css" rel="Stylesheet" href="/styles/{{ page.theme }}.css"/>
